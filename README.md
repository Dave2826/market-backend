# Market Backend

Backend project developed with Spring Boot and PostgreSQL.

## Technologies

- Java 21
- Spring Boot
- Gradle
- PostgreSQL
- JPA / Hibernate

## Current Progress

- Spring Boot project configuration
- PostgreSQL database integration
- JPA entities creation
- CrudRepository implementation
- Repository layer structure
- GitHub version control

## Database Structure

Current entities:

- Categoria
- Producto
- Compra
- CompraProducto
